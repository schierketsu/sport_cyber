# КИБЕРКАЧАЛКА

**Играй стабильно. Не выгорай.**

Прототип приложения для киберспортсменов (только **Windows**): контроль нагрузки, профилактика выгорания и тильта. Один главный индикатор и одна рекомендация — минимум действий, максимум пользы.

## Ценность продукта

- Стабильная форма и контроль нагрузки
- Сохранение мотивации и здоровья
- Не прокачка скилла и не сложная аналитика — только баланс

## Модули MVP

### 1. Индикатор риска выгорания (CORE)

- Учитывает: часы игры за день/неделю, сон, перерывы, настроение после сессии, серию поражений
- Статусы: зелёный (баланс), жёлтый (перегруз), красный (риск выгорания)
- Показывает причину и одну конкретную рекомендацию

### 2. Антитильт-режим

- Включается вручную или при плохом настроении + серия поражений
- Текст: «Сейчас ты играешь хуже не из-за скилла. Хочешь остановиться и сохранить рейтинг?»
- Варианты: 10 мин перерыв, физическая разрядка, разбор ошибок, завершить сессию

### 3. Мини-планировщик нагрузки

- Настройки: учёба/работа (часы), цель по сну, макс. время игры в день
- На главном экране: «безопасный лимит» и предупреждение при приближении к лимиту

### 4. Wellness-паузы

- Правило 20-20-20 для глаз + таймер 20 сек
- Упражнения: кисти, шея, спина (описание + таймер 30–60 сек)
- Напоминание о паузе после N минут игровой сессии

## Оверлей поверх всех окон

Приложение запускается как **компактное окно поверх всех** (always on top):

- Небольшой размер (360×300), не перекрывает игру.
- Один экран: индикатор, рекомендация, старт/стоп сессии, настроение, тильт, ссылки на планировщик и wellness.
- Кнопка **«Развернуть»** увеличивает окно до 800×600 с полным интерфейсом; кнопка **«В оверлей»** в шапке возвращает компактный вид.

## Стек прототипа

- **Flutter** — приложение для Windows
- **Provider** — состояние
- **shared_preferences** — настройки и история дней/сессий
- **flutter_local_notifications** — напоминания о wellness-паузах
- **window_manager** — для Windows: размер окна и «поверх всех окон»

Без API, без базы знаний, без сложной аналитики.

## Установка и запуск

1. Flutter SDK 3.9.2 или выше, Visual Studio с нагрузкой «Разработка классических приложений на C++».

2. Зависимости:
```bash
flutter pub get
```

3. Запуск (только Windows):
```bash
flutter run -d windows
```
